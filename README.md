# Tour Travel Website

Full Stack Tour & Travel Booking Website.

## Tech Stack
Frontend: React (Vite) + Tailwind  
Backend: Laravel API  
Database: MySQL  
Server: XAMPP

## Project Structure

tour-system-ui → React frontend  
tour-travel-backend → Laravel backend

## How to Run

### Backend
1. Move backend to XAMPP htdocs
2. Run:
composer install
php artisan migrate
php artisan serve

### Frontend
cd tour-system-ui
npm install
npm run dev

## Features
- Tour package browsing
- Booking requests
- Enquiry form
- Admin dashboard
- Admin tour management
- Booking management
- Enquiry management
